Software

A variety of softwares were used to create the 3D model of the Diligencia Basin shown on this website. Below is a brief description of each one and a link to where you can find them.

QGIS

QGIS is a free and open-source desktop geographic information system application used for viewing, editing, and analysis of geospatial data. For this project, QGIS was used to compose a geographical map of the area.

Software: https://www.qgis.org/en/site/

Blender

Blender is a professional, free and open-source 3D computer graphics software used for creating a multitude of graphics projects.

Software: https://www.blender.org/

SketchUp

SketchUp is a 3D modeling computer program for a wide range of drawing applications including but not limited to architecture, video game design, and mechanical engineering.

Software: https://www.sketchup.com/

x3dom

x3dom is an open-source framework for integrating and manipulating 3D graphics into a webpage. This was used to integrate the 3D model of NYC Buildings for this website for ease of viewing and navigating (helps pull up the data).

Software: https://www.x3dom.org/
